# Python Basics

This reading covers the fundamental concepts of Python programming.

## Variables and Data Types

Python supports several built-in data types:
- Integers: `x = 5`
- Floats: `y = 3.14`
- Strings: `name = "Alice"`
- Booleans: `is_valid = True`

## Functions

Define functions using the `def` keyword:

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

## Control Flow

Use `if`, `elif`, and `else` for conditional logic:

```python
age = 18
if age >= 18:
    print("Adult")
else:
    print("Minor")
```

## Loops

Python supports `for` and `while` loops:

```python
for i in range(5):
    print(i)

while x < 10:
    x += 1
```
